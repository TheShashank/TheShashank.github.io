About Me

I love many things - quizzing, geography, math, squash, cricket, building apps, self-improvement, listening to podcasts and so on.

It's awesome and really fun. I most recently implemented Recurrent Neural Nets for text summarization. I'm currently whizzing through Fast.AI, a course I highly recommend.

I also built an algorithm using machine learning that helps to diagnose Retinopathy of Prematurity - a curable disease that leads to blindness in infants if not corrected within an exceedingly short window. Diagnosis must be done within a couple of weeks of birth, but this is often not possible in rural India due to both a lack of equipment and a lack of specialists. I did this using Image Classification techniques. It's a huge problem and I want to fix it mainly because it's criminal to leave it like that.
